ts2es
======
By Nicholas Humfrey <njh@aelius.com>
   Falei Luo        <falei.luo@gmail.com>

ts2es is a simple tool to extract ES from a MPEG-2 Transport Stream.

Usage:

    ts2es [options] <infile> <outfile>
      -h             Help - this message.
